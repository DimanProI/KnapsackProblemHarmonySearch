Описание параметров для файла params.txt
======================================

Файл должен содержать 7 строк, каждая из которых задаёт определённый параметр. 
Параметры должны быть указаны в строгом порядке, описанном ниже.

Формат файла params.txt:
1. max_weight
2. hms
3. hmcr
4. par
5. max_iter
6. vary_param
7. vary_values

Описание параметров:
===================

1. max_weight
   - Тип: Положительное число.
   - Описание: Максимальный вес, который может выдержать рюкзак. Ограничивает суммарный вес выбранных предметов.
   - Примечание: Если max_weight меньше веса всех предметов в items.txt, алгоритм выдаст предупреждение о невозможности нахождения допустимого решения.

2. hms
   - Тип: Положительное целое число.
   - Описание: Размер гармонической памяти (Harmony Memory Size) — количество решений, хранимых алгоритмом для дальнейшей оптимизации.
   - Примечание: Большие значения увеличивают разнообразие решений, но замедляют выполнение.

3. hmcr
   - Тип: Дробное число от 0 до 1.
   - Описание: Вероятность выбора значения из гармонической памяти (Harmony Memory Considering Rate).
   - Примечание: Высокие значения ускоряют сходимость, но могут привести к застреванию в локальных оптимумах.

4. par
   - Тип: Дробное число от 0 до 1.
   - Описание: Вероятность корректировки выбранного значения (Pitch Adjusting Rate).
   - Примечание: Увеличение par способствует выходу из локальных оптимумов, но может замедлить сходимость.

5. max_iter
   - Тип: Положительное целое число.
   - Описание: Максимальное количество итераций алгоритма.
   - Примечание: Большее количество итераций улучшает качество решения, но увеличивает время выполнения.

6. vary_param
   - Тип: Строка.
   - Описание: Имя параметра, который будет варьироваться для анализа сходимости. Допустимые значения: "hms", "hmcr", "par", "max_iter".
   - Примечание: Алгоритм выполняется для каждого значения, указанного в vary_values, с выбранным параметром, изменяя его значение.

7. vary_values
   - Тип: Список чисел, разделённых пробелами.
   - Описание: Значения, которые будет принимать параметр, указанный в vary_param, для построения графиков сходимости.
   - Примечание: Для hms и max_iter значения должны быть целыми. Для hmcr и par — дробными от 0 до 1. Список должен содержать хотя бы одно значение.
